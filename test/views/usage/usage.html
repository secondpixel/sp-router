Usage

Content selector setup.

First add div to your page and add id or class:

<div id="app"><div>

Then add content to router configuration.

var router = new SPRouter({
    content: "#app"
});

Setup default route

To setup default route add defaultRoute parameter to the configuration.

var router = new SPRouter({
    content: "#app",
    defaultRoute: "/"
});

Create routes

Create basic routes

var router = new SPRouter({
    content: "#app",
    defaultRoute: "/"
    routes: [
        {
            title: "Home",
            route: "/",
            template: "views/home/home.html"
        }
    ],
});

To create a basic route you must add routes array with objects into router configuration.

All routes must have by default: title, route and template.

Additional if you want to add custom js file for every route provide script key for every route.

var router = new SPRouter({
    content: "#app",
    defaultRoute: "/"
    routes: [
        {
            title: "Home",
            route: "/",
            template: "views/home/home.html",
            script: "views/home/home.js"
        }
    ],
});

Create routes with segments

If we want to add page with products and preview page for every product we must provide some id, title etc.. and after to take this dynamic parameters in every page. Next following code will show you how to configure dynamic routes with parameters.

var router = new SPRouter({
    content: "#app",
    defaultRoute: "/"
    routes: [
        {
            title: "Products",
            route: "/products",
            template: "views/products/products.html",
            script: "views/products/products.js"
        },
        {
            title: "Product",
            route: "/products/:productId/view",
            template: "views/products/product.html",
            script: "views/products/product.js"
        }
    ],
});

All dynamic parameters you want to configure in routes must be sperated with (:).

To get this dynamic paramter (id, title etc...) you must take from router object.

var id = router.segment("productId");

Configuration

Create routerConfig.js file in your project.

Then add following code:

var router = new SPRouter({
    content: "#app",
    routes: [
        ...
    ],
    defaultRoute: ...
});

After configuration file is created add to your project after router-js script.

<script src="sp-router.js"></script>
<script src="routerConfig.js"></script>
