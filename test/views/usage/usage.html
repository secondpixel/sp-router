Usage

Content selector setup.

First add div to your index page and add selector by id or class:

<!doctype html>
<html lang="en">

<head>
    <title>Your Page</title>
    <meta content="http://localhost:3000/" name="base_url" />
</head>

<body>
    <nav class="....">
        ...
    </nav>

    <div class="container">
        <div id="app"></div>
    </div>

    <footer>
        ...
    </footer>

    <script src="js/sp-router/sp-router.js"></script>
    <script src="js/routerConfig.js"></script>
</body>

</html>

Then add content to router configuration.

var router = new SPRouter({
    content: "#app"
});

Setup default route

To setup default route for your application add defaultRoute parameter to the configuration.

var router = new SPRouter({
    content: "#app",
    defaultRoute: "/"
});

Create routes

Create basic routes

To create a basic route you must add routes array with objects into router configuration.

All routes must have by default: title, route and template.

var router = new SPRouter({
    content: "#app",
    defaultRoute: "/"
    routes: [
        {
             title: "Home",
             route: "/",
             template: "views/home/home.html"
        }
    ],
});

Additional if you want to add custom js file for every page then provide script key for every route.

var router = new SPRouter({
    content: "#app",
    defaultRoute: "/"
    routes: [
        {
            title: "Home",
            route: "/",
            template: "views/home/home.html",
            script: "views/home/home.js"
        }
    ],
});

Create routes with segments

If we want to add page with products and preview page for every product we must provide some id, title etc.. and after to get this dynamic parameters in every page. Next following code will show you how to configure dynamic routes with parameters.

var router = new SPRouter({
    content: "#app",
    defaultRoute: "/"
    routes: [
        {
            title: "Products",
            route: "/products",
            template: "views/products/products.html",
            script: "views/products/products.js"
        },
        {
            title: "Product",
            route: "/products/:productId/view",
            template: "views/products/product.html",
            script: "views/products/product.js"
        }
    ],
});

All dynamic parameters you want to configure in routes must be sperated with color(:).

Example: :productId

To get this dynamic paramter (id, title etc...) you must take from router object.

var router = new SPRouter({...});
var id = router.segment("productId");
