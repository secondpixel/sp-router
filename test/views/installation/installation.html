Installation

Via npm:

npm i sp-router-js

Implementation

Base Path

To make this router work you must set base path. To set base url path to your website application, add the following meta tag:

<meta content="http://example.com/" name="base_url" />

Include to project

From node_modules:

<script src="node_modules/sp-router-js/dist/sp-router.js"></script>
<script src="node_modules/sp-router-js/dist/sp-router.min.js"></script>

Custom path:

<script src="path/to/router/sp-router.js"></script>
<script src="path/to/router/sp-router.min.js"></script>

Configuration

Create routerConfig.js file in your project.

Then add the following code:

var router = new SPRouter({
    ...
});

After the configuration file is created, add the highlighted script to your project after router-js script.

<script src="sp-router.js"></script>
<script src="routerConfig.js"></script>
